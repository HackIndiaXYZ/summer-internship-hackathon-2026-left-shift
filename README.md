# 🌟 About VerifyX

**VerifyX** is a cutting-edge protocol designed to become the global standard for tamper-proof academic credentials.

By leveraging modern web technologies, AI-powered validation, and decentralized verification infrastructure, VerifyX enables:

- 🏛️ **Universities** to issue secure and trusted credentials
- 🎓 **Students** to truly own and control their achievements
- 🏢 **Organizations** to instantly verify records with confidence

VerifyX creates a transparent, privacy-first, and fraud-resistant ecosystem for academic verification.

---

# 🚀 Key Features

### 🔐 Decentralized Credential Verification
Tamper-proof and cryptographically verifiable academic records powered through decentralized verification infrastructure.

### 🤖 AI-Powered Validation
Integrated AI validation using **Genkit** and **Google GenAI** to detect inconsistencies and strengthen verification workflows.

### 🎨 Modern Neo-Minimal UI
A premium dark-themed interface built with:
- Framer Motion animations
- Geometric layouts
- Tailwind CSS styling
- Smooth interactive experiences

### 🛡️ Wallet-First Authentication
MetaMask-only secure access ensuring identity ownership and Web3-native authentication.

### 📂 Persistent Workflow Storage
MongoDB Atlas-backed storage for credential workflows, requests, analytics, and application state management.

### ♿ Accessible Design System
Built using **Radix UI** for accessibility-focused and production-grade user interactions.

# 🚦 Getting Started

Follow these steps to run VerifyX locally.

---

## 📋 Prerequisites

Make sure you have:

- **Node.js v20+**
- npm or yarn
- MetaMask browser extension installed

---

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone <repository-url>
cd web3
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Environment Variables

Create the following files:

#### Frontend
```bash
Frontend_verifyX/.env.local
```

#### Backend
```bash
Backend_verifyX/.env
```

Add your:
- MongoDB credentials
- OCR configuration
- VCRegistry API keys
- AI/Genkit configuration

---

### 4️⃣ Run Development Server

```bash
npm run dev
```

---

### 5️⃣ Start AI/Genkit Server (Optional)

```bash
npm run genkit:dev
```

---
